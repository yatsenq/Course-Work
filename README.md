# Детектор фейкових новин для українських новин 🇺🇦🤖

Інтелектуальна веб-система для автоматичного виявлення фейків та тематичної класифікації українських новин за допомогою Machine Learning та Deep Learning (mBERT).

## 🌟 Можливості
- **Бінарна класифікація:** Визначає, чи є стаття `TRUE` (правдивою) або `FAKE` (фейком) з високою впевненістю.
- **Класифікація теми:** Автоматично відносить новину до однієї з 4 тем: Політика, Спорт, Бізнес і Технології.
- **Гібридна AI-модель:** Використовує ансамбль із **mBERT** (Multilingual BERT) та класичної **Логістичної регресії** з TF-IDF.
- **Автоматичні докази:** Підсвічує конкретні маркери маніпуляцій (ключові слова) у тексті.
- **Звітність:** Генерує детальні PDF-звіти для кожного аналізу.
- **Особистий кабінет:** Повна історія перевірок для авторизованих користувачів.
- **Панель адміністратора:** Статистика в реальному часі та управління користувачами.

## 🛠 Технічний стек
- **Backend:** Python, Flask, Flask-Login, SQLAlchemy
- **AI/ML:** PyTorch, Transformers (Hugging Face), Scikit-learn
- **NLP:** PyMorphy3 (для морфології української мови)
- **База даних:** PostgreSQL / SQLite
- **Деплоймент:** Docker, Hugging Face Spaces

## 🏗 Архітектура системи
Система використовує **розподілену модель деплойменту** для оптимізації ресурсів:
1. **Репозиторій моделей:** Зберігає важкі ваги BERT та векторизовані моделі.
2. **App Space:** Запускає Flask веб-додаток у Docker-контейнері.
3. **ML Service:** Динамічно завантажує ваги при старті та виконує інференс ансамблем із 4 моделей.

## 🚀 Встановлення та локальний запуск
1. Клонувати репозиторій:
   ```bash
   git clone https://github.com/yatsenq/Course-Work.git
   cd Course-Work/Deployment/flask_app
   ```
2. Встановити залежності:
   ```bash
   pip install -r requirements.txt
   ```
3. Налаштувати змінні середовища:
   Створіть файл `.env` з `SECRET_KEY` та `DATABASE_URL`.
4. Запустити додаток:
   ```bash
   python app.py
   ```
